Enunciado

Escreva uma formatação em HTML para uma página de agenda idêntica à da figura.

A página deve ter uma estrutura válida, tendo apenas um elemento de topo html, um elemento head e um elemento body. O título da página deve ser "Agenda".

O único elemento dento do body deve ser um formulário que quando submetido irá invocar o programa agendar.cgi localizado na mesma directoria desta interface.

Dentro do formulário os vários elementos são posicionados com uma tabela. As células da tabela são separadas entre si por 3 pixel. O conteúdo das células está afastado do seu bordo em 5 pixel. A cor de fundo da tabela é lightgrey e a cor do título é lightBlue.

Os campos de texto para tarefas têm 20 caracteres de comprimento. Note ainda que os símbolos "<" e ">" são obtidos com "<" e ">", respectivamente. Os botões de navegação que usam estes símbolos, assim como os botões de rodapé ("Aceitar" e "Cancelar") são objectos com tipo button.

Note ainda que não deve definir os atributos name e value nos objectos gráficos do formulário, com excepção dos botões anteriormente referidos (navegação e rodapé).

A página deve ter uma aparência análoga à imagem seguinte.
